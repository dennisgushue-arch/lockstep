# Lockstep Creative System

## Core Message

### pressure → action → consequence

Use this exact flow everywhere:

- Ads trigger pressure
- Landing creates action
- Paywall commits consequence

---

## App Store + Landing Alignment

### Headline

**STOP SAYING IT. DO IT.**

### Subhead

**Follow through—or lose.**

### Promise

Lockstep makes it cost you.

### Final CTA line

**Your word should mean something.**

---

## Paywall System

### Top

- You followed through.
- Most people don’t.

### Core

- Free lets you try this.
- Pro makes it real.

### Pressure Levers

- Multiple active pacts
- Witness Mode (real accountability)
- Hard Mode (no easy outs)
- Stronger consequences

### CTA

- Primary: **TURN UP THE PRESSURE**
- Secondary: **Stay limited (1 pact only)**

### Trigger Rules

1. After first win
2. When creating second pact
3. After recovery

### Don’ts

- Don’t lead with pricing
- Don’t list generic premium features
- Don’t over-explain

---

## Ad Creative System (5–10 sec)

### Style Rules (all ads)

- Black background
- White text
- Red for impact only
- Fast cuts
- 5–10 seconds max

### Messaging Rule

Every ad must answer:

**What happens if I don’t follow through?**

### Ad 1 — Direct Hook

1. YOU SAID YOU WOULD
2. But you didn’t
3. Now it costs you
4. CTA: Download Lockstep

### Ad 2 — Relatable

1. “I need to start working out”
2. Still didn’t do it?
3. Show app UI lock-in screen
4. CTA: Download Lockstep

### Ad 3 — Consequence

1. YOU DIDN’T DO IT
2. Score drops
3. That’s the difference
4. CTA: Download Lockstep

### Ad 4 — Transformation

1. Said it 10 times
2. Did it 0 times
3. After Lockstep: Did it 3 times
4. CTA: Download Lockstep

---

## System Fit

- **Ads:** emotional interruption and tension
- **Landing:** clear system explanation
- **Paywall:** commitment conversion point

Lockstep is not selling productivity. Lockstep is selling consequence + accountability.
