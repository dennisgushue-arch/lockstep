# Data Deletion Policy for Lockstep

**Effective date:** 2026-03-26

This page explains how users can request deletion of account and personal data associated with Lockstep.

## 1. Current Deletion Method

Lockstep currently supports verified deletion requests through support channels.

If an in-app self-serve delete option is added later, this policy will be updated to include exact in-app steps.

## 2. Web/Email Deletion Request (required fallback)

If in-app deletion is unavailable or inaccessible, users can request deletion by:

- Emailing **[support@lockstep.app](mailto:support@lockstep.app)** with subject: `Data Deletion Request - Lockstep`
- Including account identifier(s): your account email and any relevant user ID

We may request verification of identity before processing.

## 3. Deletion Scope

Upon successful and verified request, we delete or anonymize:

- Account profile data
- User-generated content associated with the account (e.g., commitments, reflections, journal entries, voice note records), unless retention is legally required
- Connected integration metadata linked to account records where feasible

## 4. Retained Data (if applicable)

We may retain limited information for legal, security, fraud prevention, tax/accounting, or regulatory obligations.

## 5. Processing Time

We aim to process verified deletion requests within **30 days** (or sooner where required by applicable law).

## 6. Contact

For deletion requests and status updates:

- **Email:** [support@lockstep.app](mailto:support@lockstep.app)
- **Privacy:** [privacy@lockstep.app](mailto:privacy@lockstep.app)
- **Company:** Lockstep
